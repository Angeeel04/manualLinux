CHMOD

El comando CHMOD se utiliza para dar permisos a archivos.

Parámetros:

-x

Quitar permisos de ejecucion

-r

Quitar permisos de lectura

-w

Quitar permisos de escritura

Otros parámetros:

-v

Muestra los permisos que tenía antes y los que tiene despues de utilizarlo

-R, --recursive

Aplica los permisos a todos los subdirectorios / archivos dentro de directorios de manera recursiva

-c

como verbose pero informando sólo cuando se hace un cambio

Ejemplos:

Otorgar permisos de lectura y escritura a un archivo.

chmod +rw myfile.txt

-rw-r--r-- 1 root root 0 Apr 5 16:31 myfile.txt

Información detallada a la hora de cambiar permisos

chmod -v 755 archivo1.txt

El modo de 'archivo1.txt' cambia de 0644 (rw-r--r--) a 0755 (rwxr-xr-x)
